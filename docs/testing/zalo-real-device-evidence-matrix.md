## 3. 证据要求

- 原始录屏、截图、Zalo 版本管理页面、脱敏日志和审计导出放在限制访问、带版本/不可变保留策略的证据库；仓库只记录 URI、SHA-256、执行时间和结论。
- 截图/录屏必须遮盖登录二维码、完整手机号、头像等非必要个人信息、access/phone token、App Secret、CLI 登录文件和签名 URL。
- 网络证据只保留 method、脱敏 path、HTTP 状态、correlation ID 和时间；不得保存认证 Header 或原始 Zalo 响应。
- 每条 `PASS` 至少有能证明设备/版本和结果的证据。发现问题后记录工单、修复 commit 和复测证据，不覆盖第一次失败事实。
